# Security Policy

## Official Channels

The official Clarium channels for this landing are only the ones explicitly listed on the website and in this repository.

Official references:

- Website domain configured in production
- Official landing repository
- Official app repository
- Official author LinkedIn
- Official contact email

## APK Distribution

Do not trust APK files published outside the official Clarium website or official channels listed there.

Before installing:

1. Confirm the domain is the official one.
2. Confirm the repository links match the official GitHub account.
3. Avoid mirrors, reuploads, “modded” APKs or shortened links from third parties.

## Reporting Fraud or Security Issues

If you find:

- fake websites
- fake repositories
- repackaged APKs
- impersonation of the author
- security vulnerabilities in this landing

Report through the official contact email listed on the website.

## Scope

This repository covers the official Clarium landing page only.

If the mobile app repository is public but still missing a software license, do not assume third parties are allowed to redistribute or repackage it.
